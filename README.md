# Konrad-Fischer-Archiv

Dieses Repository ist ein **statisches Archiv der veröffentlichten Texte von Konrad Fischer**  
(ursprünglich: konrad-fischer-info.de).

Ziel des Projekts ist die **dauerhafte Sicherung, Durchsuchbarkeit und Zitierfähigkeit** der Inhalte
in einem offenen, wartbaren Format – unabhängig von der ursprünglichen Website-Struktur.

---

## Motivation

Die Originalseite basiert auf historisch gewachsenem HTML, ist technisch fragil
und nur eingeschränkt durchsuchbar.

Dieses Projekt verfolgt daher folgende Ziele:

- langfristige Archivierung
- verlustfreie Überführung von HTML → Markdown
- saubere interne Verlinkung
- Nutzung moderner statischer Dokumentations-Tools
- vollständiger Verzicht auf Tracking, Zähler oder externe Abhängigkeiten

---

## Inhalt

- mehrere hundert Artikel (HTML → Markdown konvertiert)
- Originalstruktur weitgehend erhalten
- Inhalte unverändert übernommen (keine inhaltliche Redaktion)
- Metadaten wie Titel und ursprüngliche URLs sind rekonstruierbar

**Hinweis:**  
Dieses Repository enthält ausschließlich bereits öffentlich zugängliche Texte
zu Archivierungs- und Dokumentationszwecken.

---

## Technischer Aufbau

- **Quellformat:** Markdown (`.md`)
- **Struktur:** thematisch gegliederte Verzeichnisse
- **Sidebar / Navigation:** manuell gepflegt
- **Darstellung:** erfolgt über ein separates Docsify-Projekt  
  (nicht Bestandteil dieses Repositories)

Dieses Repository enthält **keine laufzeitabhängige Logik** und ist vollständig statisch.

---

## Suche

Die Suche wird **nicht über GitHub** bereitgestellt.

Für lokale oder gehostete Versionen wird ein **vorgefertigter Suchindex**
(außerhalb dieses Repos) verwendet, da clientseitige Volltextsuche
bei mehreren hundert langen Dokumenten nicht performant ist.
